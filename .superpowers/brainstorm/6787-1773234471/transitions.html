C. Row Expand

La fila seleccionada se expande hasta llenar toda la pagina, y su contenido se transforma en el form. Muy visual, conecta lista con detalle.

+ Wow factor, conexion visual directa
− Mas complejo de implementar, requiere View Transitions API o FLIP
Mi recomendacion: B (Crossfade)
Para "simple, buena UX, sin necesidad de explicar": el crossfade es el mas sutil y rapido. Se implementa con CSS puro (200-300ms opacity + scale). No distrae, no se siente pesado, y es lo que usan Notion y Linear. El slide (A) es bueno tambien pero puede sentirse "app movil". El row expand (C) es impresionante pero complejo y fragil.
